# aide-memory in GitHub Copilot

**Status today:** rules template only. `aide-memory init` writes a curated rules file for Copilot. Hooks and MCP config aren't generated for Copilot yet.

If you add aide-memory as an MCP server manually in Copilot's MCP config, the seven MCP tools work like they do in Claude Code or Cursor. Beyond that, deeper integration may come based on user feedback. To request it, [open an issue](https://github.com/aide-memory/aide-memory/issues).
