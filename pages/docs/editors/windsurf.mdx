# aide-memory in Windsurf

**Status today:** rules template only. `aide-memory init` writes a curated rules file for Windsurf. Hooks and MCP config aren't generated for Windsurf yet.

If you add aide-memory as an MCP server manually in Windsurf's MCP config, the seven MCP tools work like they do in Claude Code or Cursor. Beyond that, deeper integration may come based on user feedback. To request it, [open an issue](https://github.com/aide-memory/aide-memory/issues).
