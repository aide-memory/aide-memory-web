# aide-memory in Codex

**Status today:** rules template only. `aide-memory init` writes a curated rules file for Codex. Hooks and MCP config aren't generated for Codex yet.

If you add aide-memory as an MCP server manually in Codex's MCP config, the seven MCP tools work like they do in Claude Code or Cursor. Beyond that, deeper integration may come based on user feedback. To request it, [open an issue](https://github.com/aide-memory/aide-memory/issues).
